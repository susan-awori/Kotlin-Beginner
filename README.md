# Prompt-Powered Kickstart: Beginner Toolkit for Kotlin GUI

**Course:** Moringa AI Essentials – Capstone Project  
**Assignment:** Build a beginner-friendly toolkit for a new technology using AI prompts  

---
## Overview

This project is designed for beginners to learn **Kotlin**, a modern JVM language, by creating a simple **Hello World program**. It demonstrates:

- Setting up a Kotlin project in IntelliJ IDEA
- Writing a basic Kotlin program
- Running the program to see output in the console

---

## Technology Used

- **Kotlin** – JVM-based programming language  
- **IntelliJ IDEA** – IDE for Kotlin development  
- **JDK 21** – Java Development Kit required to run Kotlin programs  

---

## System Requirements

- Windows / macOS / Linux  
- IntelliJ IDEA Community or Ultimate Edition  
- Kotlin plugin installed in IntelliJ  
- JDK 17 or 21 installed on your system  

---

## Installation & Setup

1. Download and install **IntelliJ IDEA** from [JetBrains](https://www.jetbrains.com/idea/download/).  
2. Install **Kotlin plugin** via **Settings → Plugins → Marketplace → Kotlin**.  
3. Install **JDK 17 or 21** (Temurin recommended) and note the installation path.  
4. Open IntelliJ → **File → New Project → Kotlin → JVM | Empty Project**.  
5. Set **Project SDK** to your installed JDK (17 or 21).
